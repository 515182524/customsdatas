# 海关贸易分析方法

## 核心指标

- 总金额：有效金额列求和。
- 环比：`本期 / 上期 - 1`。首期不计算。
- 同比：`本期 / 上年同期 - 1`。缺少上年同期不计算。
- 份额：分类金额 / 总金额。
- CR3 / CR5：金额排名前 3 / 5 的份额之和。
- HHI：各分类金额份额平方和乘以 10,000。用于描述集中度，不直接等同于竞争结论。
- 金额变化贡献：末期金额减首期金额，按分类拆解。
- 加权单价：有效数量记录对应金额合计 / 有效数量合计。只在同一数量字段的单位唯一且数量大于零时计算。
- 重量单价：金额 / 重量数量，例如美元/千克。
- 数量单价：金额 / 件数、台数或个数，例如美元/台。
- 月度单价变异系数：月度加权单价的总体标准差 / 月度均值，用于描述波动程度。
- 双数量比值：第二数量合计 / 第一数量合计，例如千克/台，用于描述平均重量或规格。

## 分析顺序

1. 先判断时间跨度、金额币种和进出口口径。
2. 再看总体规模与月度变化。
3. 分析国家/地区、商品、国内区域、贸易方式。
4. 做首末期变化拆解，找增长和下降来源。
5. 对每个有效数量口径，分析数量、加权单价、月度波动和主要市场单价。
6. 做交叉维度分析，例如贸易伙伴 × 商品、贸易伙伴 × 注册地。
7. 最后写机会、风险与数据限制。

## 图表使用

| 问题 | 首选图表 | 使用条件 |
|---|---|---|
| 金额随时间如何变化 | 折线图 | 至少 2 个时期 |
| 哪些分类贡献最大 | 横向条形图 | 至少 2 个分类 |
| 谁推动了首末期变化 | 正负变化条形图 | 至少 2 个时期 |
| 主要伙伴在各月如何分布 | 热力图 | 至少 2 个时期和 2 个伙伴 |
| 数量与加权单价如何变化 | 两张折线图 + 联动表 | 数量单位唯一且至少 2 个时期 |
| 平均规格或重量如何变化 | 双数量比值折线图 | 两个数量字段都有效 |
| 哪些主要市场单价较高 | 条形图 | 仅比较主要金额市场，避免小样本极端值主导 |
| 两类节点之间的金额流向 | 桑基图 | 节点数量受控且流向语义明确 |
| 构成占比 | 条形图或矩形树图 | 类别较多时不要使用饼图 |

## 结论边界

- 描述“数据表现出什么”，不要无证据解释“为什么”。
- 最新月份可能未完结时，把下降写成待验证信号。
- 只覆盖单一商品时，重点放在市场、区域与时间，不虚构产品组合结论。
- 没有原产地或目的国列时，报告只能分析贸易伙伴。
- 没有价格、税率、企业或运输方式字段时，不做对应结论。
- 加权单价变化可能由产品规格和市场结构变化造成，不直接视为真实报价变化。
- 两个数量口径分别计算，不把“台”和“千克”等不同单位直接相加或互换。
- 双数量比值变化可能来自产品组合变化，不直接视为单个产品规格发生变化。
